📄 README.md for Task 1: Digital Literacy Infographic
Project Overview

Task 1: Create a Digital Literacy Awareness Infographic This task involved creating a one-page visual resource to help batchmates understand the importance of digital literacy and the essential tools required for academic success.

Course: CSE0001 - Digital Literacy 


Module: Module 1 

Task Description
In my role as a Student Digital Ambassador, I designed an infographic that covers the following core areas of digital literacy:


Definition of Digital Literacy: What it means to be digitally literate in 2026.


Essential Digital Tools: Key platforms every VIT student should know.


Safe Internet Practices: Tips for staying secure and responsible online.

Tools Used

Design Tool: [Canva / Prezi / Miro] 


Asset Type: One-page visual infographic 

Reflections (Preview for Report)

Design Choice: I chose a [clean/vibrant/professional] layout to ensure the information is easily digestible for first-year students.


Key Learning: While creating this, I realized how important it is to balance visual appeal with factual accuracy to keep the audience engaged.
